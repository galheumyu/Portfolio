# Gitee Pages 配置文件
# 访问地址：https://gitee.com/galheumyu/Portfolio/pages
# 或者：https://galheumyu.gitee.io/portfolio

# Gitee Pages 使用说明：
# 1. 在 Gitee 创建仓库，导入 GitHub 的 Portfolio
# 2. 进入仓库 → 服务 → Gitee Pages
# 3. 选择部署分支：master 或 main
# 4. 选择部署目录：/
# 5. 点击启动
# 6. 等待审核通过（首次需要实名认证）

# 注意：Gitee Pages 对免费用户需要实名认证才能开启
